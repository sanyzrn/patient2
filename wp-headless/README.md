# پل Headless چت‌بات نفس

این پوشه شامل فایل پلی است که به **پورتال React** اجازه می‌دهد از افزونهٔ چت‌بات وردپرس به‌صورت cross-origin استفاده کند.

معماری: پورتال صاحب **رابط کاربری و برندینگ** چت است؛ وردپرس فقط **مغز AI + ذخیرهٔ فرم‌ها + اعلان‌ها** را فراهم می‌کند.

## نصب (یک‌بار، روی سایت وردپرس)

۱. فایل `nafas-chatbot-headless.php` را در این مسیر سایت وردپرس کپی کن:

```
wp-content/mu-plugins/nafas-chatbot-headless.php
```

> اگر پوشهٔ `mu-plugins` وجود ندارد، بسازش. افزونه‌های mu خودکار فعال می‌شوند (نیازی به فعال‌سازی دستی نیست).

۲. داخل فایل، مقدار `NAFAS_HEADLESS_ORIGINS` را به دامنهٔ پورتال خودت تنظیم کن (مثلاً `https://patient.nafaspharmed.com`).

## این فایل چه می‌کند
- هدرهای **CORS** را فقط برای origin‌های مجاز روی اکشن‌های AJAX چت‌بات اضافه می‌کند.
- درخواست **preflight (OPTIONS)** را پاسخ می‌دهد.
- یک endpoint عمومی `nafas_chatbot_nonce` می‌سازد که یک **nonce تازه** به پورتال می‌دهد تا بتواند چت/فرم/امتیاز را امن فراخوانی کند.

## سمت پورتال
- آدرس وردپرس در `src/config.ts` → `VITE_WP_BASE_URL` (پیش‌فرض `https://nafaspharmed.com`).
- `connect-src` در `public/.htaccess` اجازهٔ تماس به دامنهٔ وردپرس را دارد.

## نکات
- شناسهٔ محصولات پورتال باید با شناسهٔ محصولات افزونه یکی باشد تا پایگاه دانش (KB) درست انتخاب شود.
- کلیدهای API و توکن‌های بله/تلگرام **فقط روی وردپرس** می‌مانند؛ هرگز در پورتال قرار نمی‌گیرند.
